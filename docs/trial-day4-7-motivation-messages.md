# 무료 체험 4~7일차 동기부여 문구 설계

> 작성일: 2026-03-11
> 목적: 구독 관리 페이지 — 4~7일차 실천 패턴별 문구 정의
> 관련 문서: `free-trial-user-types.md`, `trial-day1-3-motivation-messages.md`

---

## 1~3일차와의 구조적 차이

1~3일차에는 구독 관리 페이지에 오는 유형이 거의 E(안심 확인)뿐이었기 때문에 "참여 상태"만으로 분기할 수 있었다. 4~7일차에는 A, B, C, D 네 유형이 모두 방문하며, 같은 참여 데이터라도 방문 목적이 다르다. 다만 서버에서 방문 목적을 알 수 없으므로, 판별 가능한 변수 조합으로 유형을 추정한다.

---

## 유형별 방문 시점 교차 분석

| 체험 시점 | participation | 추정 유형 | 방문 목적 |
|---|---|---|---|
| 4~5일 | 0 | B (충동적 등록자) | "아 그거 등록했었지" — 아직 해지 결심 전 |
| 6~7일 | 0 | B 또는 D (해지 예정자) | 해지 버튼 찾기 |
| 3~5일 | 1~2 | C (비교 쇼핑객) 또는 B | 가성비 판단 / 이탈 고민 |
| 4~5일 | 1~3 | A (진지한 탐색자) 초기 | 습관 형성 중 |
| 5~7일 | 3+ | A (진지한 탐색자) | 다음 단계 확인 |
| 5~6일 | any | D (해지 예정자) | 해지 실행 |
| 5~7일 | 1~3 | C (비교 쇼핑객) | 최종 비교 판단 |

## 4~7일차 전체 패턴 분류

| 패턴 | 조건 | 상황 설명 | 추정 유형 |
|---|---|---|---|
| ⑨ 미참여 중반 | trialDay 4~5, participation=0 | 등록 후 한 번도 실천 안 함 (중반) | B |
| ⑩ 미참여 후반 | trialDay 6~7, participation=0 | 결제일 임박, 한 번도 실천 안 함 | B, D |
| ⑪ 저참여 쉬는 중 | participation 1~2, !practicedToday | 1~2일 해보고 멈춤 | B, C |
| ⑫ 늦은 시작 | participation=1, streak=1, practicedToday | 4일차 이후 첫 실천 | B |
| ⑬ 쉬다 돌아옴 | participation≥2, streak=1, practicedToday | 쉬었다가 다시 실천 | A, B |
| ⑭ 연속 실천 중 | streak 2~3, practicedToday | 습관 형성 구간 | A |
| ⑮ 장기 연속 실천 | streak≥4, practicedToday | 습관이 자리잡은 상태 | A |
| ⑯ 꾸준한 실천자 쉬는 중 | participation≥3, !practicedToday | 3일+ 실천했지만 오늘은 아직 | A, C |

## 패턴별 동기부여 문구

### ⑨ 미참여 중반 (trialDay 4~5, participation=0)

```
아직 경험하지 못한 솔루션이 기다리고 있어요.
지금 시작해도 충분합니다.
```

**CTA:** "첫 번째 솔루션 시작하기"

**설계 의도:** 추정 유형 B. "아 그거 등록했었지"하고 떠올린 시점. 아직 해지를 결심하지 않았기 때문에 재진입 메시지가 효과적이다. "0일"을 보여주면 이탈이 확정되므로 숫자를 노출하지 않는다.

---

### ⑩ 미참여 후반 (trialDay 6~7, participation=0)

```
체험 기간이 얼마 남지 않았어요.
15분이면 충분합니다.
```

**CTA:** "솔루션 경험해보기"

**설계 의도:** 추정 유형 B 또는 D. 결제일이 임박해 해지 버튼을 찾으러 왔을 가능성이 높다. 이 사용자를 강제로 잡으려 하면 안 되지만, 마지막으로 한 번 경험할 기회를 가볍게 제시한다. 전체 패턴 중 유일하게 시간 언급("얼마 남지 않았어요")을 사용하되, 구체적 숫자("N일 남음")는 사용하지 않는다.

---

### ⑪ 저참여 쉬는 중 (participation 1~2, !practicedToday)

```
지난번 실천이 아직 몸에 남아 있어요.
오늘도 15분, 이어가 볼까요?
```

**CTA:** "오늘의 솔루션 시작하기"

**설계 의도:** 추정 유형 B 또는 C. 1~3일차 ⑥과 동일한 프레임이지만, 경과일이 길어졌으므로 "몸에 남아 있다"는 표현이 더 의미 있다. 참여 이력이 적으므로 숫자를 노출하지 않는다.

---

### ⑫ 늦은 시작 (participation=1, streak=1, practicedToday)

```
첫 걸음을 내디뎠습니다.
시작한 날이 1일차예요.
지금부터가 {userName}님의 여정입니다.
```

**CTA:** "오늘의 솔루션 이어보기"

**설계 의도:** 추정 유형 B가 뒤늦게 참여한 경우. 1~3일차 ⑦과 동일한 리프레이밍. 체험 경과일과 무관하게 "시작한 날이 1일차"라는 메시지로 개인 기준의 진행 감각을 제공한다.

---

### ⑬ 쉬다 돌아옴 (participation≥2, streak=1, practicedToday)

```
바쁜 하루 속에서도 다시 찾아주셨어요.
지금 이어가면 충분합니다.
```

**CTA:** "이어서 솔루션 보기"

**설계 의도:** 추정 유형 A 또는 B. 1~3일차 ⑤와 동일한 문구. 7일 체험에서 "쉬었다"보다 "바쁜 하루 속에서도 다시 찾아왔다"는 공감이 자연스럽다.

---

### ⑭ 연속 실천 중 (streak 2~3, practicedToday)

**streak = participationDays일 때:**
```
{streak}일 연속, 몸이 달라지고 있어요.
꾸준함이 변화를 만듭니다.
```

**streak ≠ participationDays일 때:**
```
웰니스 루틴이 만들어지고 있어요.
꾸준함이 변화를 만듭니다.
```

**CTA:** "오늘의 솔루션 이어보기"

**설계 의도:** 추정 유형 A. 동적 숫자 미노출 원칙 적용 — streak와 상장의 웰니스 실천일(participationDays)이 동일할 때만 숫자를 표시. 불일치 시 "루틴 형성" 톤으로 전환하여 숫자 없이 꾸준함을 인정. 실천 후 방문자의 자기효능감과 소유감(psychological ownership)에 공명하는 메세지.

---

### ⑮ 장기 연속 실천 (streak≥4, practicedToday)

**streak = participationDays일 때:**
```
{streak}일 연속 실천.
{userName}님의 웰니스 루틴이 자리잡고 있어요.
```

**streak ≠ participationDays일 때:**
```
꾸준한 실천이 루틴이 되고 있어요.
{userName}님의 웰니스 루틴이 자리잡고 있어요.
```

**CTA:** "오늘의 솔루션 이어보기"

**설계 의도:** 추정 유형 A. 동적 숫자 미노출 원칙 적용. 4일 이상 연속은 습관이 "자리잡은" 상태이므로, 숫자 대신 "루틴이 되고 있다"는 질적 전환을 강조. 사용자 이름으로 개인화.

---

### ⑯ 꾸준한 실천자 쉬는 중 (participation≥3, !practicedToday)

```
{participation}일의 실천이 쌓여 있어요.
오늘도 이어가 볼까요?
```

**CTA:** "오늘의 솔루션 시작하기"

**설계 의도:** 추정 유형 A 또는 C. ⑪(저참여 쉬는 중)과의 차이는, 참여 이력이 3일 이상이므로 숫자를 보여줘도 긍정적이라는 점. 축적된 실천일을 인정하면서 오늘도 이어가도록 유도한다.

---

## 분기 로직 우선순위 (4~7일차)

```
1. streak≥4 AND practicedToday                  → ⑮ 장기 연속
2. streak 2~3 AND practicedToday                → ⑭ 연속 실천 중
3. participation≥2, streak=1, practicedToday     → ⑬ 쉬다 돌아옴
4. participation=1, streak=1, practicedToday      → ⑫ 늦은 시작
5. participation≥3, !practicedToday              → ⑯ 꾸준한 실천자 쉬는 중
6. participation 1~2, !practicedToday            → ⑪ 저참여 쉬는 중
7. participation=0, trialDay 6~7                 → ⑩ 미참여 후반
8. participation=0, trialDay 4~5                 → ⑨ 미참여 중반
```

---

## 1~3일차(①~⑧)와의 차이 요약

| 차이 | 1~3일차 | 4~7일차 |
|---|---|---|
| 미참여 처리 | 단일 패턴 (①) | 시점별 2단계 분리 (⑨⑩) |
| 연속 실천 | 2일/3일 고정 (③④) | streak 수를 동적 표시 (⑭⑮) |
| 쉬는 중 | 단일 패턴 (⑥) | 참여도에 따라 2단계 분리 (⑪⑯) |
| 톤 변화 | "시작 / 습관 형성" | "꾸준함 / 루틴 확립" |
| 카운트다운 | 완전 배제 | ⑩에서만 "얼마 남지 않았어요" 사용 |

---

## 공통 톤앤매너 원칙 (1~3일차와 동일)

1. **카운트다운 금지:** "N일 남았습니다"는 사용하지 않는다. (⑩만 예외적으로 "얼마 남지 않았어요" 사용)
2. **신체 감각 중심:** "몸이 달라진다", "몸에 남아 있다" — 힐에코의 요가/웰니스 맥락에 맞는 표현.
3. **짧은 호흡:** 한 줄당 15자 내외. 모바일 화면에서 한눈에 읽히는 길이.
4. **판단 배제:** "0일", "빈 상태" 같은 부정적 데이터를 직접 노출하지 않는다.
5. **개인화:** `{userName}`을 핵심 문구에 삽입하되 남용하지 않는다 (⑫, ⑮에서만 사용).
6. **동적 숫자:** 4~7일차에서는 `{streak}`, `{participation}`을 문구에 직접 삽입한다 (3일 이상일 때만).
