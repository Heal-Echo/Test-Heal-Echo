# 무료 체험 1~3일차 동기부여 문구 설계

> 작성일: 2026-03-11
> 목적: 구독 관리 페이지 Wellness Certificate 영역 — 1~3일차 실천 패턴별 문구 정의
> 관련 문서: `free-trial-user-types.md`, `similar-service-trial-day1-3-strategy.md`

---

## 판별 변수

| 변수 | 설명 | 출처 |
|---|---|---|
| `participationDays` | 총 실천 일수 | `/api/user/practice-record` |
| `consecutiveDays` | 현재 연속 실천일 | `/api/user/practice-record` |
| `daysElapsed` | 체험 시작 후 경과일 | 결제 정보 기반 계산 |

---

## 1~3일차 전체 패턴 분류

| 패턴 | 조건 | 상황 설명 |
|---|---|---|
| ① 미실천 | participation=0 | 아직 한 번도 실천하지 않음 |
| ② 첫 실천 | participation=1, consecutive=1, elapsed=1 | 체험 1일차에 첫 실천 완료 |
| ③ 2일 연속 | consecutive=2 | 2일 연속 실천 |
| ④ 3일 연속 | consecutive=3 | 3일 연속 실천 |
| ⑤ 쉬고 돌아옴 | participation≥2, consecutive=1 | 실천 후 쉬었다가 다시 실천 |
| ⑥ 쉬는 중 | participation≥1, consecutive=0 | 실천한 적 있으나 오늘은 아직 |
| ⑦ 늦은 첫 시작 | elapsed≥2, participation=1, consecutive=1 | 2일차 또는 3일차에 처음 실천 |
| ⑧ 늦은 시작+2일 연속 | elapsed=3, participation=2, consecutive=2 | 늦게 시작했지만 2일 연속 중 |

---

## 패턴별 동기부여 문구

### ① 미실천 (participation=0)

```
오늘이 시작이에요.
하루 15분, 첫 번째 솔루션이 기다리고 있습니다.
```

**CTA:** "첫 번째 솔루션 시작하기"

**설계 의도:** "0일"이라는 숫자를 직접 보여주지 않는다. `free-trial-user-types.md` 유형 B 분석 — "0일 시청 데이터를 보여주면 이탈이 확정된다"는 시사점 반영.

---

### ② 첫 실천 완료 (participation=1, consecutive=1, elapsed=1)

```
첫 걸음을 내디뎠습니다.
몸이 기억하기 시작했어요.
내일도 15분, 이어가 볼까요?
```

**CTA:** "오늘의 솔루션 이어보기"

**설계 의도:** 첫 실천의 의미를 강조하되 부담을 주지 않는다. "몸이 기억하기 시작했어요"는 힐에코의 웰니스 맥락(요가, 자율신경)에 맞는 신체 감각 중심 표현.

---

### ③ 2일 연속 실천 (consecutive=2)

```
몸이 달라지고 있어요.
습관은 이렇게 만들어집니다.
내일도 이어가면, 작은 리듬이 생겨요.
```

**CTA:** "오늘의 솔루션 이어보기"

**설계 의도:** 동적 숫자 미노출 원칙 적용. 상장의 "웰니스 실천일" 숫자와 불일치할 수 있는 연속일 텍스트를 제거. "리듬"이라는 단어로 습관 형성을 자연스러운 과정으로 프레이밍.

---

### ④ 3일 연속 실천 (consecutive=3)

```
꾸준한 실천.
이제 몸이 이 시간을 기다리기 시작합니다.
{userName}님만의 웰니스 루틴이 만들어지고 있어요.
```

**CTA:** "오늘의 솔루션 이어보기"

**설계 의도:** 동적 숫자 미노출 원칙 적용. "꾸준한 실천"으로 연속일 숫자 없이 성취를 인정. "몸이 기다린다"는 표현으로 내적 동기(신체 감각)를 강화. 사용자 이름을 넣어 개인화.

---

### ⑤ 쉬었다가 돌아온 실천 (participation≥2, consecutive=1)

```
바쁜 하루 속에서도 다시 찾아주셨어요.
지금 이어가면 충분합니다.
```

**CTA:** "이어서 솔루션 보기"

**설계 의도:** 7일 무료 체험에서 "쉬었다"는 의도적 휴식의 뉘앙스가 있지만, 실제로는 일상에 밀려 시간이 훅 지나간 경우가 대부분이다. "바쁜 하루 속에서도"로 공감하고, 돌아온 행동 자체를 긍정하여 유형 B(충동적 등록자)의 재이탈을 방지.

---

### ⑥ 실천한 적 있으나 오늘은 아직 (participation≥1, consecutive=0)

```
지난번 실천이 아직 몸에 남아 있어요.
오늘도 15분, 이어가 볼까요?
```

**CTA:** "오늘의 솔루션 시작하기"

**설계 의도:** "아직 안 했다"는 판단 대신 "지난번이 남아 있다"는 긍정적 프레임. 가벼운 톤으로 재참여를 유도하되 압박하지 않는다.

---

### ⑦ 늦은 첫 시작 (elapsed≥2, participation=1, consecutive=1)

```
첫 걸음을 내디뎠습니다.
시작한 날이 1일차예요.
지금부터가 {userName}님의 여정입니다.
```

**CTA:** "오늘의 솔루션 이어보기"

**설계 의도:** 체험 시작일과 실천 시작일의 차이에서 오는 "늦었다"는 감정을 해소. "시작한 날이 1일차"라는 리프레이밍으로 개인 기준의 진행 감각을 제공.

---

### ⑧ 늦은 시작 + 2일 연속 (elapsed=3, participation=2, consecutive=2)

③과 동일한 문구를 사용한다.

```
이틀 연속, 몸이 달라지고 있어요.
습관은 이렇게 만들어집니다.
내일이면 3일 — 작은 리듬이 생겨요.
```

**설계 의도:** 늦게 시작했더라도 연속 실천의 가치는 동일하다. 경과일이 아닌 연속일 기준으로 문구를 결정하므로 별도 분기 불필요.

---

## 분기 로직 우선순위 (코드 구현 시 참고)

```
1. consecutive=3         → ④ 3일 연속
2. consecutive=2         → ③ 2일 연속 (⑧ 포함)
3. participation≥2, consecutive=1 → ⑤ 쉬고 돌아옴
4. participation=1, consecutive=1, elapsed=1 → ② 첫 실천
5. participation=1, consecutive=1, elapsed≥2 → ⑦ 늦은 첫 시작
6. participation≥1, consecutive=0 → ⑥ 쉬는 중
7. participation=0        → ① 미실천
```

---

## 공통 톤앤매너 원칙

1. **카운트다운 금지:** "N일 남았습니다"는 사용하지 않는다. 압박이 아닌 격려가 목적.
2. **신체 감각 중심:** "몸이 기억한다", "몸이 달라진다", "몸에 남아 있다" — 힐에코의 요가/웰니스 맥락에 맞는 표현.
3. **짧은 호흡:** 한 줄당 15자 내외. 모바일 화면에서 한눈에 읽히는 길이.
4. **판단 배제:** "아직 안 했다", "0일", "빈 상태" 같은 부정적 데이터를 직접 노출하지 않는다.
5. **개인화:** `{userName}`을 핵심 문구에 삽입하되 남용하지 않는다 (④, ⑦에서만 사용).
